# Fine-Tuning-YOLO26-Models-Custom-Dataset

This repository contains two Jupyter notebooks for fine-tuning YOLO26 on custom datasets. The first notebook fine-tunes YOLO26 for construction site PPE (personal protective equipment) detection, and the second fine-tunes YOLO26 to recognize the American Sign Language (ASL) alphabet. Both notebooks can be executed on Google Colab, Kaggle, or a local machine. Proper instructions have been given in the notebooks.

It is part of the LearnOpenCV blog post - [How to Fine-Tune YOLO26 for Safety Gear and Sign Language Detection](https://learnopencv.com/how-to-fine-tune-yolo26-for-safety-gear-and-sign-language-detection/).

[![How to Fine-Tune YOLO26 for Safety Gear and Sign Language Detection](./YOLO26_Finetuning_Thumbnail_image.jpg)](https://learnopencv.com/how-to-fine-tune-yolo26-for-safety-gear-and-sign-language-detection/)


## What You Will Learn

- How to get a free Roboflow API key and pull a ready-made dataset in YOLO format
- What fine-tuning is, and why a pretrained model alone cannot detect your own classes
- How to fine-tune YOLO26 on custom data, evaluate it, and run it on images and video
- How to build a construction-site PPE safety checker (hard hat, vest, mask) with a live violation alert
- How to train YOLO26 to read the American Sign Language (ASL) alphabet from a single hand
- How the very same workflow transfers from one problem to a completely different one
